How to run the program:
1) install python3 locally
2) install python extention on VS code
3) Open command promt on VS code and type:
   python3 server1.py
4) Open another command promt on VS code and type:
   python3 client.py

Extra: 
1) IP: Your local ipv4
2) Port: 7568
